# Hair Salon Booking App (MERN Stack)

A modern, responsive hair salon booking platform built with the **MERN stack (MongoDB, Express.js, React.js, Node.js)**. This application allows users to book appointments, browse services, and interact with salon staff, while providing admins full control over bookings, services, and client management.

---

## 🌐 Live Demo
[Click here to view the live app](https://mern-hairsalonapp.onrender.com)

---

## Features

- 🔒 **JWT Authentication** — Secure user login and session management.
- 🧭 **Responsive Design** — Fully mobile-friendly UI built with React & Bootstrap.
- 🛍️ **Service Booking** — Users can browse services, select staff, and book appointments.
- 📅 **Booking Management** — Users can view, cancel, or reschedule appointments.
- ⚡ **Admin Dashboard** — Manage services, appointments, and staff efficiently.
- 🧩 **Real-Time Updates** — Booking status and admin updates reflected immediately using Axios.
- 💬 **Contact/Chat** — Users can send inquiries or messages to the salon directly.

---

## Project Workflow

1. **User Authentication**: Users register, log in, and manage sessions securely using JWT.  
2. **Service Browsing**: Users can explore salon services, filter by category, or select preferred stylists.  
3. **Booking Management**: Book appointments in real-time and view upcoming/rescheduled bookings.  
4. **Admin Panel**: Add, update, or remove services, manage bookings, and analyze client activity.  
5. **Notifications & Chat**: Users receive updates on bookings; real-time chat connects users to admin for inquiries.
