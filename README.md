# Hair Salon Booking App (MERN Stack)

A responsive hair salon booking platform built with the **MERN stack (MongoDB, Express.js, React.js, Node.js)**. Users can browse services, book appointments, and admins can manage services and bookings.

---

## 🌐 Live Demo
You can access the live version of this project here:

HairSalon BookingApp:https://mern-hairsalonapp-1.onrender.com

---

## Features

- 🧭 **Responsive Design** — Mobile-friendly UI with React & Bootstrap  
- 🛍️ **Service Booking** — Browse services and book appointments  
- ⚡ **Admin Dashboard** — Add, update, or remove services and manage bookings

---

## Technology Stack

| Category      | Technologies                             |
|---------------|------------------------------------------|
| Frontend      | React.js, Redux, HTML5, CSS3, Bootstrap |
| Backend       | Node.js, Express.js                       |
| Database      | MongoDB (Mongoose)                        |
| Deployment    | Render                                   |

---

## Project Workflow

1. **Service Browsing**: Explore salon services by category or stylist  
2. **Booking Management**: Book and manage appointments  
3. **Admin Panel**: Manage services and track bookings
